El propósito del sitio es para brindar información contextual sobre las etiquetas básicas que componen al lenguaje estructural de las páginas web HTML con sus respectivos atributos y valores.

Encontrará una sección donde se abordan las temáticas, otra sección donde podrá visualizar contenido multimedia relacionado y sección para actividades de aprendizaje.

El estudio de cada una de las secciones le permitirá adquirir un conocimiento básico, no se pretende ahondar en los conceptos y elementos de una página web. Es importante mencionar que se estudia las estructuras del lenguaje HML5.
