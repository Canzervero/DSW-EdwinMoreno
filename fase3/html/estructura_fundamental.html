ESTRUCTURA FUNDAMENTAL DE HTML

Las páginas web estan construidas mediante lenguaje HTML el cual se encuentra en su versión 5. El acrónimo HTML se refiere a HyperText Markup Language, que en español se podría traducir como Lenguaje de Marcado de Hypertexto. No es un lenguaje de programación en sí mismo, pero es una especie de Lenguaje y este es interpretado por navegadores.

Para que un navegador pueda interpretar adecuadamente una página web, esta debe tener una estructura fundamental que está compuesta por las siguientes etiquetas:

<!DOCTYPE html>
<html lang="en">
<head></head>
<body></body>
</html>

Estructura fundamental en editor de código. Imagen Propia.

La etiqueta <!DOCTYPE html>, indica al navegador web que el código que va a interpretar es HTML5

Las etiquetas <html lang="en"></html>, indican al navegador web que todo lo que están dentro de ellas es el código HTML5

Las etiquetas <head></head>, indican al navegador web que todo lo que está dentro de ellas son archivos de código css, javascript entre otras configuraciones

Las etiquetas <body></body>, indican al navegador web que todo lo que está dentro de ellas corresponde al código HTML que contiene la información que va a ser presentada en la página web
