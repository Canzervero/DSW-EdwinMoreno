ESTRUCTURA SEMÁNTICA DE HTML

Gracias al HTML5, las páginas web tienen un lenguaje semántico, es decir, dispone de etiquetas que indican qué tipo de contenido almacenan, por ejemplo: encabezado, pie de página, barra de navegación, articulos, formularios, etc.

<header></header>
<nav></nav>
<article></article>
<footer></footer>

Estructura semántica de un sitio web. Imagen tomada de: www.ediciones-eni.com

Las etiquetas <header></header>, le indica al navegador web que el contenido es el encabezado del sitio web

Las etiquetas <nav></nav>, le indican al navegador web que el contendio corresponde al menú de navegación del sitio web

Las etiquetas <article></article>, le indican al navegador web que el contenido corresponde a un artículo del sitio web

Las etiquetas <footer></footer>, le indican al navegador web que el contenido es el pie de página del sitio web
