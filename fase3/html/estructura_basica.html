ESTRUCTURA BÁSICA DE HTML

Además de las etiquetas ya vistas, existen otras que son básicas dado que con ellas se puede construir una página web elemental que permita presentar la información de forma adecuada. A continuación, se describen dichas etiquetas:

<h1></h1>
<p></p>
<a></a>
<img></img>

Estructuras básicas de HTML5. Imagen propia

Las etiquetas <h1></h1>, se utilizan para los títulos del sitio web; se pueden utilizar <h2><h3><h4><h5><h6>, cada una de estas, hacen que los títulos sean de menor y mayor tamaño

Las etiquetas <p></p>, se utilizan para crear los párrafos de texto que serán mostrados en el sitio web

Las etiquetas <a></a>, se utilizan para crear los enlaces o Links hacia otros recursos del sitio web

Las etiquetas <img></img>, se utilizan para insertar las imágenes que serán utilizadas en el sitio web

Hasta ahora, hemos visto las etiquetas fundamentales, semánticas y básicas, además de estas etiquetas, existen muchas otras, sin embargo estas son las más comunes. Adicionalmente, todas las etiquetas cuentan con atributos que son transversales y otros que son específicos. A continuación, se relacionan los transversales:

id="", se utiliza para identificar de manera única a cada etiqueta

class="", se utiliza para asignar las clases css para darle estilo y diseño a cada etiqueta

name="", se utiliza para darle un nombre específico a cada etiqueta

href="", se utiliza en la etiqueta <a> para enlazar un recurso y src="" se utiliza para asignar una imagen en la etiqueta <img>
