CSS GRID LAYOUT

De acuerdo con la w3schools, el CSS Grid Layout ofrece un sistema de capas basado en grilla o rejilla, con filas y columnas, lo cual hace más fácil el diseño de páginas web sin tener que usar flotadores o posicionamiento:

<div class="grid-container">
<div class="grid-item">1</div>
<div class="grid-item">2</div>
<div class="grid-item">3</div>
</div>

CSS GRID LAYOUT. Tomada de: https://www.w3schools.com/css/css_grid.asp

Las etiquetas <div class="grid-container"></div>, se utilizan para declarar que se utilizará el sistema GRID LAYOUT; se incluye clase CSS que permite trabajar bajo este esquema

Las etiquetas <div class="grid-item">1</div>, se utilizan para crear cada una de las celdas que conformarán a la grilla

Se requiere el siguiente código CSS .grid-container { display: grid; }, para que se pueda utilizar el esquema Grid Layout
